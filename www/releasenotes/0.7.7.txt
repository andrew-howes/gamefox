### About this version

#### Version 0.7.7

Two more issues caused by the GameFAQs update have been fixed.

#### Version 0.7.6 (2010-05-09)

Many more fixes have been made for GameFAQs' new URL scheme.

spynae's GFPastel 2010 stylesheet has also been updated.

#### Version 0.7.5 (2010-05-05)

[SpotFAQs](http://www.takatocss.co.cc/spotfaqs.xml) is replaced with SpotFAQs extras.
This replacement CSS is designed to be used with the SpotFAQs skin on GameFAQs
for styling GameFOX features.

Post number in quote header is now clickable.
It links to the quoted post in the same topic.

New GameFAQs theme:
[GFPastel 2010](http://www.gamefaqs.com/boards/genmessage.php?board=565885&topic=53000033)
by spynae.
You can view its description and enable it from the Style options (Tools |
GameFOX | Style).

Highlighting now properly obeys the highlighting group ranks.
Priority will be given to the first matching group.
